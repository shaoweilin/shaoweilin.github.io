Instructions

To use the software, right click on the link to FrontLeg above to download the Windows executable file. You may also want to download the four sample files above in the same way. Double click on "frontleg.exe" to start the program. Go to "File > Open" and click on the sample files to view the knots. Click on the various functions under "View" and "Tools" to compute some knot invariants. To create your own knots, use SnapPea, because FrontLeg accepts files saved by SnapPea. Alternatively, create a file with the following text file format (also available from the SnapPea website): LinkProjectionFileFormat.txt.
